## Project Details

- **GitHub Repository**: [https://github.com/shohag-cse-knu/express-react-crud-app.git](https://github.com/shohag-cse-knu/express-react-crud-app.git)

## Features

- **User List**: Shows all Users.
- **Add User**: Adding user with name and email.
- **Edit User**: Updating user with name and email.
- **Delete Product**: Deleting an user.

## Prerequisites

- **Express**: Express 5.2.1
- **React**: React 18.3.1
- **NodeJS**: v24.12.0
- **PostgreSQL**: PostgreSQL 17.7

## Setup Instructions

### 1. Clone the Repository
```bash
git clone https://github.com/shohag-cse-knu/express-react-crud-app.git
````
### 2. Navigate to backend
```bash
cd backend
````
### 3. Install npm
```bash
npm install
````
### 4. Create a PostgreSQL Database 'crud_app' or Run db.sql in PGAadmin

### 5. Create .env and update DB_Name and DB_Pass
```bash
cp .env.example .env
````
### 6. Run Express Project
```bash
node server.js
````
### 7. Navigate to React Project
```bash
cd frontend
````
### 8. Install npm
```bash
npm install
````
### 9. Run React Project
```bash
npm start
````

## Technologies Used

- **Frontend**: ReactJS
- **Backend**: Express REST API, PostgreSQL
